# 景品識別　プロトタイプ
ターゲットデバイスはMaixCam
YOLOv5/v8で試作
カスタムデータセットはカーゾック吉祥寺店にてプレーして獲得した景品４種各１個
## 使うツール
- **MaixVision**：MaixCamへのDeployやファイルコピーなど
- **MaixHub**：初期のデータセットの作成とアノテーション、その後はオフラインでtrain
- **MaixCam**：ターゲットデバイスですが、カスタムデータセットの画像を撮影するためにも使う
